# [level 1] 약수의 합 - 12928

### 회고
굳이 n 까지 for 루프를 돌릴 필요 없이, n 의 제곱근까지만 돌리면 된다.
ex) n = 36 일때, 다음과 같이 대칭이 된다.

1*36 = 36   대칭
2*18 = 36
3*12 = 36
4*9 = 36
============
6*6 = 36 - 자기 자신
============
9*4 = 36    대칭
12*3 = 36
18*2 = 36
36*1 = 36

때문에, for 루프를 1 ~ n 의 제곱근까지만 돌리고 6*6 처럼 중복된 약수가 더해지는 케이스만 제외 해주면 된다.
